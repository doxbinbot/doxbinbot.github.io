[Doxbinbot](https://doxbinbot.github.io/doksbot/) - это мощный инструмент, предназначенный для сбора и предоставления конфиденциальной информации о людях (доксинга). Бот ищет данные о физических и юридических лицах из различных источников, включая публичные базы данных, социальные сети и другие открытые источники информации. DoxBot обеспечивает доступ к такой информации с целью осведомления и использования в различных сферах, таких как кибербезопасность, правоохранительные органы, расследования и т. д. Важно использовать DoxBot в соответствии с законодательством и этическими нормами.
